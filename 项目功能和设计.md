# 镜头流 (LensFlow) 小程序开发指令 (Prompt)

**目标**：请根据以下规格说明，为我构建一个面向“独立摄影师”的轻量级 CRM 小程序前端原型。

## 1. 视觉风格指南 (UI Style)

- **设计语言**：极致简约、画廊感、高端摄影画册风格。
- **配色方案**：深色模式（背景：#121212，文字：#FFFFFF/E0E0E0）。
- **交互细节**：使用毛玻璃效果（Glassmorphism）的底栏，图片切换要有平滑的淡入淡出动效。
- **字体**：系统默认细体，配合衬线体标题。

## 2. 核心页面结构

### 页面一：摄影师管理端首页 (Dashboard)

- **顶部栏**：展示摄影师 Logo、店名及极简数据面板（今日拍摄、待修图数）。
- **金刚区**：四个大卡片按钮（风格管理、档期预约、项目订单、交付中心）。
- **活动流**：纵向时间轴展示客户动态（如：某客户已选风格、某客户已付定金）。
- **悬浮按钮 (FAB)**：右下角一个 `+` 号按钮，点击弹出“生成风格确认链接”。

### 页面二：客户侧-风格确认落地页 (Moodboard Selection)

- **Cover Page**：满屏大图，带有“向下滑动开启审美之旅”的引导提示。
- **沉浸式列表**：
  - **风格组卡片**：每组包含 1-2 张代表作，非对称排版。
  - **选择逻辑**：点击图片即选中，再次点击取消。右上角显示优雅的勾选图标。
  - **计数限制**：底部固定栏显示“已选 n 组风格”，并有确认提交按钮。
- **全屏详情交互**：点击风格组卡片可进入该风格的详情横滑图库。

## 3. 技术栈建议

- **框架**：Uni-app 或微信原生开发。
- **样式**：使用 Tailwind CSS 或 SCSS 快速构建样式。
- **后端（模拟）**：先使用本地 JSON 数据模拟风格库和项目列表。

## 4. Claude 第一步任务描述

> "请先帮我实现**客户侧-风格确认落地页**。要求：
>
> 1. 首页是一个满屏的沉浸式封面。
> 2. 向下滚动后进入风格列表，每组风格由一个大标题和两张错位排列的图片组成。
> 3. 点击图片时，图片边缘要有一个呼吸灯效果的边框或勾选标识。
> 4. 请使用 Vue (Uni-app) 或原生微信小程序代码编写，并确保 UI 极其接近高端摄影 App 的质感。"
